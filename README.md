## üöÄ Key Features
- **Automated Ledger Logic:** Synchronizes supply with real-time demand to optimize stock levels.
- **Order Fulfillment Tracking:** A streamlined workflow managing the lifecycle of an order from entry to delivery.
- **Mobile-Responsive UI:** Built with React and Bootstrap for cross-device accessibility.
- **Data Integrity:** Utilizes PostgreSQL with atomic transactions for accurate inventory logging.
